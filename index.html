Get started with Imaculate-UI

Quickly and easily get started with Immaculate's compiled, production-ready files with this barebones example featuring some basic HTML and helpful links. Download all our examples from github.

Download From Github

css/immaculate-styles.css   &   js/immaculate-scripts.js.

Quick Start

For a fast and simple roleout of our styles, we reccomend using the links below in your html file to get started.

CSS

Copy-paste the stylesheet <link> into your <head> before all other stylesheets to load our CSS.

<link rel="stylesheet" href="https://immaculate-ui.web.app/css/immaculate-styles.css">

JS

At the bottom of your <body> copy-paste the <script> to load our JavaScript functionality.

<script src="https://immaculate-ui.web.app/js/immaculate-scripts.js"></script>

Immaculate Buttons

Make use of Immacluate's great variety of button styles. You can combine different styles which can let you show off immaculate buttons for your website.

Regular

To intialize the styling for your button you start with imma-button class. You can personalize your button color from available styles such as btn_black, btn_red, btn_green, btn_blue.

<button class="imma-button">Default</button>
<button class="imma-button btn_black">Black</button>
<button class="imma-button btn_red">Red</button>
<button class="imma-button btn_green">Green</button>
<button class="imma-button btn_blue">Blue</button>

Rounded

You can keep adding optional styling to the same buttom you're creating by using the round class

<button class="imma-button round">Default</button>
<button class="imma-button btn_black round">Black</button>
<button class="imma-button btn_red round">Red</button>
<button class="imma-button btn_green round">Green</button>
<button class="imma-button btn_blue round">Blue</button>

3D

Here we have buttons making use of all styles including td class

<button class="imma-button round td">Default</button>
<button class="imma-button btn_black round td">Black</button>
<button class="imma-button btn_red round td">Red</button>
<button class="imma-button btn_green round td">Green</button>
<button class="imma-button btn_blue round td">Blue</button>

Immaculate Navbar

Navbars help get projects structured and started. With our immaculate navbar, you cna kickstart your project with a navigational menu.

Regular

This navbar style is a simple squared look to give your designt he simple start it needs. Use the topnav class in the most top level dive to get started.

<div class="topnav">
<div class="itab-links">
<button class="itab">Home</button>
<button class="itab">News</button>
<button class="itab">Contact</button>
</div>
<div class="itab-links">
<button class="itab">Image</button>
</div>
</div>

Rounded

If you're looking for a more exquisite looking Navbar menu, the rounded tab will fill those needs. Add the itab-round class in the most top level div to get started.

<div class="topnav itab-round">
<div class="itab-links">
<button class="itab">Home</button>
<button class="itab">News</button>
<button class="itab">Contact</button>
</div>
<div class="itab-links">
<button class="itab">Image</button>
</div>
</div>

Immaculate Anamations

Being a member of Immaculate-UI has its perks. This animations tab holds sleek and simple animations you cna use in your build pertaining to button animations.

Previous and Next

Arrows are animated next to the name of the button.

<button class="anime-arrow prev"> <span> prev</span></button>
<button class="anime-arrow next"> <span> next</span></button>

Sumbit, Thanks

After clicking submit the button will say thanks with a checkmark.

<button id="btn-submit" class="anime-submit">
<p id="submit-text">Submit</p>
<div class="check-box">
<svg class="submit-svg" xmlns="http://www.w3.org/2000/svg" viewBox="0 0 50 50">
<path class="submit-path" fill="transparent" d="M14.1 27.2l7.1 7.2 16.7-16.8" />
</svg>
</div>
</button>
<script>
var btnSubmit = document.getElementById("btn-submit");
var submitText = document.getElementById("submit-text");
btnSubmit.onclick = function() {
submitText.innerHTML = "Thanks";
btnSubmit.classList.add("submit-active");
}
</script>

Continue

When hovering over the button it will rise and a solid boarder will appear.

<div class="anime-wrap">
<button id="anime-continue" class="anime-continue">Continue</button>
</div>
